・必要フレームワーク、ライブラリ、パッケージ、データベース,ミドルウェア
vue.js(3.2), express, typescript

front: tailwindcss, daisyui, socket.io-client, vue-router
back: nodemon, socket.io, prisma, cors, dotenv
db: mysql
middleware: express-session, express-validator, express-fileupload

・機能
ログイン画面でユーザー名アイコン画像をアップロードし、ログイン (login session作成)
ログイン画面でバリデーションを行う
login sessionが存在する場合はチャットルーム、ない場合はログイン画面に遷移
ログインしてる人たち同士でLineのようなリアルタイムグループチャットをすることができる
ログアウト機能も実装(loginセッションの破棄)
自分のチャットは、右側、他人は左側に表示、スタイリングはtailwindcss、daisyuiで行う
チャットした内容はdbに保存し永久化できる、prismaを使用し、users,postsテーブルを作成


・DB
users
	id primarykey autoincrement
	name
	image
	created_at


posts
	id primarykey autoincrement
	user_id (users.id)
	message
	created_at
